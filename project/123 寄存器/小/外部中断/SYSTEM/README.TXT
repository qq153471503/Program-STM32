    这是一个通用的STM32启动及控制代码集.它没有使用ST的库,完全是基于寄存器的操作.它包括三部分:
1,sys
    sys部分包含了STM32的系统级别的代码,包括:时钟配置,IO口输入输出管理,中断管理,待机模式设置,BKP操作,软复位,JTAG配置等.
2,usart
    usart部分包含了对串口1的初始化,用于串口调试,或者串口打印信息.
3,delay
    delay部分包含了2个基本的延时函数,delay_us和delay_ms,这两个函数采用STM32的SYSTICK作为延时定时器,所以在使用delay的时候,需要占用SYSTICK.

    这些代码理论上支持STM32F101,F102,F103,F105,及F107.但是只在F103上做了测试.
